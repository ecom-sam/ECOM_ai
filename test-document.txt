測試文檔內容

這是一份用於測試 Q/A 生成系統的文檔。

主要功能：
1. 文檔處理和解析
2. AI 問答對生成
3. 向量儲存和檢索

系統特色：
- 支援 PDF 文檔處理
- 整合 OpenAI API
- 使用 Couchbase 向量儲存
- 提供 RESTful API 接口

技術架構：
採用 Spring Boot 3.x 微服務架構，結合 Spring AI 框架實現智能問答功能。系統能夠自動從上傳的文檔中提取內容，並生成相關的問答對，幫助用戶快速理解文檔內容。

使用場景：
1. 企業知識庫建設
2. 客服系統支援
3. 學習資料整理
4. 文檔智能分析

這個系統可以大幅提升資訊檢索效率，並為用戶提供更好的學習體驗。
